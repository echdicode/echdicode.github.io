Spring Boot là gì ?

Spring framework là một framework của java nó được chia thành nhiều module, mỗi module làm một chức năng, ví dụ Spring Core, Web, Data access, AOP,… Spring được xây dựng dựa trên 2 khái niệm nền tảng là Dependency injection và AOP (Aspect Oriented Programming).

Spring Boot là một module của Spring Framework, cung cấp tính năng RAD (Rapid Application Development) – Phát triển ứng dụng nhanh. Nó giải quyết khó khăn khi sử dụng Spring framework là việc cấu hình dự án. Để tạo được một web HelloWorld thì Spring phải cấu hình rất nhiều thứ:

  • Tạo Maven hoặc Gradle project
  • Thêm các thư viện cần thiết
  • Tạo XML để cấu hình project, cấu hình các bean
  • Code và build thành file WAR
  • Cấu hình Tomcat server để chạy được file WAR vừa build

Nhưng với Spring Boot mọi thứ lại trở nên quá giản đơn:

  • Spring Boot không yêu cầu cấu hình XML
  • Có các tính năng của Spring Framework.
  • Tạo ứng dụng độc lập, có thể chạy bằng java -jar (cho cả java web)
  • Nhúng trực tiếp các ứng dụng server (Tomcat, Jetty…) do đó không cần phải triển khai file WAR
  • Cấu hình ít, tự động cậu hình bất kỳ khi nào có thể (Giảm thời gian viết code, tăng năng suất)
  • Không yêu cầu XML config…
  • Cung cấp nhiều plugin
  • Chuẩn cho Microservices (Cloud support; giảm việc setup, config; các thư viện hỗ trợ…)

Ứng dụng của Spring Boot ?

Spring boot được sử dụng để tạo ra các microservice.

Microservice là là một mô hình cho phép bạn lập trình và phát triển nhiều service (dịch vụ/ chức năng) một cách độc lập. Các service trong kiến trúc microservice là độc lập với nhau, chúng có thể có kiến trúc khác nhau, sử dụng công nghệ khác nhau hoặc thậm chí có database riêng. Chúng trao đổi thông tin với nhau thông qua môi trường mạng.
