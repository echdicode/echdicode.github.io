INVERSION OF CONTROL (IoC) Ở bài trước chúng ta đã giải quyết được vấn đề các module liên kết chặt chẽ với nhau khó sửa đổi rồi. Nhưng lại nảy sinh ra một vấn đề nữa là một chương trình đâu chỉ có một module mà còn có rất nhiều module khác cần inject nữa (cuộc sống SugarBaby đâu chỉ cần mỗi daddy đúng không nào). Nghĩ đến cảnh một module có hàng tá dependency mà bạn phải tự inject từng cái đã mệt rồi , quản lý code lúc này càng mệt hơn.

GIỚI THIỆU SPRING BOOT

echdicode published on Dec 12, 2021
Spring Boot là gì ? Spring framework là một framework của java nó được chia thành nhiều module, mỗi module làm một chức năng, ví dụ Spring Core, Web, Data access, AOP,… Spring được xây dựng dựa trên 2 khái niệm nền tảng là Dependency injection và AOP (Aspect Oriented Programming). Spring Boot là một module của Spring Framework, cung cấp tính năng RAD (Rapid Application Development) – Phát triển ứng dụng nhanh.
